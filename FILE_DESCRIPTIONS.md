# File descriptions

This file lists the main files and directories in the repository and a short description for each.

- `backend/app.py` — Flask backend application. Handles API endpoints and server logic.
- `frontend/index.html` — Minimal static frontend used to interact with the backend.
- `frontend/assets/logo.png` — Logo and static assets used by the frontend.
- `ml_engine/engine.py` — Machine learning model code (training/inference pipeline and helpers).
- `ml_engine/requirements.txt` — Python library requirements for the ML engine.
- `docs/PROJECT_REPORT.md` — Project report and documentation.
- `.github/workflows/python-ci.yml` — GitHub Actions workflow that runs basic CI checks and tests.
- `scripts/init_db.sql` — SQL used to initialize project database schema and sample data.
- `.gitignore` — Files and directories ignored by Git (virtual environments, build artifacts).
- `README.md` — Project overview and quick start instructions.
- `LICENSE` — Project license (MIT).

Note: Virtual environments like `.venv/` and `ml_engine/venv/` are not tracked and should not be committed.
